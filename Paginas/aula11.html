O que aprendi

Nessa aula, aprendi sobre media queries em CSS, técnicas que permitem criar designs responsivos e adaptáveis a diferentes tamanhos de tela. Aprendi como utilizar propriedades como @media para definir estilos condicionais e como aplicá-las a elementos HTML. Além disso, tive a oportunidade de praticar aplicando essas técnicas em meus projetos, o que me ajudou a entender melhor como as media queries podem ser usadas para melhorar a experiência do usuário em diferentes dispositivos.

O que desenvolvi

Nessa aula, consegui desenvolver a habilidade de usar media queries em CSS para criar designs mais responsivos e adaptáveis. Pratiquei o uso de propriedades como @media para definir estilos condicionais e como aplicá-las a elementos HTML. Além disso, experimentei diferentes combinações dessas propriedades para criar designs mais sofisticados e visualmente atraentes, o que me ajudou a entender melhor como as media queries podem ser usadas para resolver desafios comuns de tipografia em CSS.
