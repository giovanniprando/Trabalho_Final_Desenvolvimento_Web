Formulários HTML

Aprendendo a criar formulários em HTML para coletar informações dos usuários(Sem Back-end).

Progresso da TimeLine Aula 3 de 12

O que aprendi

Nessa aula, aprendi sobre formulários em HTML, que são elementos usados para coletar informações dos usuários. Aprendi sobre as diferentes tags e atributos disponíveis para criar campos de entrada, como textos, senhas, caixas de seleção e botões. Além disso, tive a oportunidade de praticar criando alguns formulários simples usando o VS Code, o que me ajudou a entender melhor como os formulários funcionam e como os dados são enviados.

O que desenvolvi

Nessa aula, desenvolvi um formulário simples em HTML para coletar informações dos usuários. O formulário incluía campos de entrada para nome, email e senha, além de um botão de envio. Pratiquei a estruturação do formulário usando as tags HTML apropriadas e adicionei atributos para melhorar a usabilidade, como placeholders e validação básica. Embora o formulário não tenha funcionalidade de back-end, foi uma ótima oportunidade para entender como os formulários são construídos e como os dados são organizados para envio.
