O que aprendi

Nessa aula, aprendi sobre keyframes em CSS, técnicas que permitem criar animações complexas e controles detalhados de movimento em suas páginas web. Aprendi como utilizar propriedades como @keyframes para definir animações personalizadas e como aplicá-las a elementos HTML. Além disso, tive a oportunidade de praticar aplicando essas técnicas em meus projetos, o que me ajudou a entender melhor como os keyframes podem ser usados para melhorar a experiência do usuário.

O que desenvolvi

Nessa aula, consegui desenvolver a habilidade de usar keyframes em CSS para criar animações mais complexas e controladas em minhas páginas web. Pratiquei o uso de propriedades como @keyframes para definir animações personalizadas e como aplicá-las a elementos HTML. Além disso, experimentei diferentes combinações dessas propriedades para criar designs mais sofisticados e visualmente atraentes, o que me ajudou a entender melhor como os keyframes podem ser usados para resolver desafios comuns de tipografia em CSS.
