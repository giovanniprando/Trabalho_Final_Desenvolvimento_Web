HTML Básico

Aprendendo os fundamentos do HTML e criando minhas primeiras páginas web.

Progresso da TimeLine Aula 2 de 12

O que aprendi

Nessa aula, aprendi sobre o HTML, que é a linguagem de marcação usada para criar a estrutura de páginas web. Aprendi sobre as tags HTML, que são usadas para definir diferentes elementos em uma página, como tags estruturais, textos, títulos (h1-h6), links e imagens e muito mais. Além disso, tive a oportunidade de praticar criando algumas páginas HTML usando o VS Code, o que me ajudou a entender melhor como a estrutura do HTML funciona e como os diferentes elementos interagem entre si.

O que desenvolvi

Desenvolvi habilidades em criar páginas HTML usando o VS Code, aplicando os conceitos aprendidos sobre tags HTML e estrutura de páginas web. Além disso, pratiquei a criação de elementos como cabeçalhos, parágrafos e links, consolidando meu entendimento sobre a estrutura e o funcionamento do HTML.
