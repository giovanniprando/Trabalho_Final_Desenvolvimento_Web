Descrição do Projeto

Este projeto foi desenvolvido como um portfólio acadêmico para registrar e apresentar os conhecimentos adquiridos ao longo das aulas. O site reúne conteúdos estudados, projetos desenvolvidos, desafios enfrentados e competências adquiridas durante o curso.
O objetivo é documentar a evolução do aprendizado de forma organizada, visual e interativa, utilizando tecnologias de desenvolvimento web.
